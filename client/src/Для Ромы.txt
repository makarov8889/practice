FrontEnd:

Все маршруты находяться в файле utils/consts.js

Index.js:
В этом файле я полдключаю Context для того что бы в дальнейшем его использовать для хранения данных, как его использовать

// Код (./pages/auth/Auth):
import { useContext } from "react";
const { user } = useContext(Context);
console.log(user);

// Если хотим получить вменяемые данные то 
import { toJS } from "mobx";
const userData = toJS(user.user);
console.log("Данные пользователя -", userData.token);

Конструктор нашего Context user находится в store/userStore


Что бы они сохранились нужно переключаться между страницами так что бы сайт не перезагружался, работает это следующим образом

// Код (./pages/auth/Auth):
import { useNavigate } from "react-router-dom";
const navigate = useNavigate();
navigate(MAIN_ROUTE);


App.js:
Здесь подключаются элементы которые всегда будут находиться на странице как Navbar, а AppRouter упровляет роутами (маршрутизация)


component/AppRouter.js:
Здесь роуты поделены на те что показываються всем пользователям, и на те что показываються только авторизованым


В папке http нахоядться файлы которые обращаються к серверу и получает его токен, из токена достаються адекватные данные через

// код http/index.js
config.headers.authorization = `Bearer ${localStorage.getItem("token")}`;
